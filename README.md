# Task 4 - Flask REST API

This project is a simple REST API built using Flask.

## Features

- Get all users (GET)
- Get single user (GET)
- Add new user (POST)
- Update user (PUT)
- Delete user (DELETE)

## Technologies Used

- Python
- Flask
- Postman

## How to Run

1. Install Flask
   pip install flask

2. Run the app
   python app.py

3. Open in browser
   http://127.0.0.1:5000/users

## Concepts Used

- REST API
- HTTP Methods
- JSON
- Status Codes
- CRUD Operations
